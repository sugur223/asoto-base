# asotobase（あそとベース）

## プロジェクト概要

asotobaseは、個人の「あそと」な活動を促進し、コミュニティ内での共創・共栄を実現するプラットフォームです。

### 「あそと」とは

**あそと**を磨く3つの要素：
- **①関係性** - 人とのつながり、ネットワーキング
- **②多動性** - 主体的な行動、チャレンジ
- **③感受性** - 内省、気づき、学び

会社では「組織性・一貫性・論理性」が磨かれやすい一方で、asotobaseではこれを補完する「関係性・多動性・感受性」を育み、**「仕事があそびになる」**状態を目指します。

## ビジョン

- **あそび → あそと**：趣味や興味から主体的な活動へ
- **しごと → あそと**：仕事をあそびのように楽しむ状態へ

この2つの相互作用により、人生全体が充実した「あそと」状態になることを支援します。

## 主な機能

### 1. 個人のあそと/自律を促進（②多動性 ③感受性）
- **あそとステップの可視化**：行動のステップをゲーミフィケーション的に表示
- **ネクストアクション整理**：自分とシステムで次の行動を明確化
- **内省ログページ**：個人専用の振り返りスペース
- **AIコーチング**：行動の振り返りと気づきを促すAIサポート

### 2. 共創/共栄しやすくなる（①関係性）
- **スマートマッチング**：個人の情報から人・機会・企業をマッチング
- **全体進捗の可視化**：あそと全体の動きを可視化
- **貢献度ポイント制**：asotobaseへの貢献を可視化し、共存共栄を体感
- **コミュニティ活性化**：イベントやプロジェクトの情報を見やすく整理

## 解決する課題

### 現状の課題
- あそとは「重要だが緊急でない」領域のため、多忙時に時間・気力を確保できない
- 行動の一歩目が重く、継続が難しい
- 行動後の内省が不十分でやりっぱなしになる
- Discord等でのやりとりが認識しづらい
- 社外の仕事のご縁を体感しづらい（関係性の重要性が見えにくい）
- マッチングが属人的
- ビジョンや進捗が定性情報のみで可視化されていない

### asotobaseでの解決
これらの課題に対し、システマティックな可視化・ゲーミフィケーション・AIコーチングを組み合わせることで、継続的な「あそと」活動を支援します。

## 開発環境セットアップ

詳細なセットアップ手順は [SETUP.md](SETUP.md) を参照してください。

### クイックスタート

```bash
# リポジトリをクローン
git clone <repository-url>
cd asoto-base

# Docker Composeでサービス起動
docker-compose up -d

# データベースマイグレーション
docker-compose exec backend alembic revision --autogenerate -m "Initial migration"
docker-compose exec backend alembic upgrade head
```

- フロントエンド: http://localhost:3000
- バックエンドAPI: http://localhost:8000/docs

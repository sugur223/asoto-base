# asotobase - コンセプト・ビジョン

## 1. あそとの哲学

### あそと状態とは

**あそと**は、「**あそび**（遊び）」と「**しごと**（仕事）」を組み合わせた造語です。

仕事と遊びの境界が溶け合い、人生全体が主体的で充実した状態を指します。

```
あそび（趣味・興味）
    ↓
  あそと（主体的な活動）
    ↑
しごと（労働）
```

この双方向の相互作用により、**「仕事が遊びになる」**状態を実現します。

### あそとを構成する3つの要素

| 要素 | 説明 | 具体例 |
|------|------|--------|
| **①関係性** | 人とのつながり、信頼関係の構築 | ネットワーキング、コラボレーション、メンタリング |
| **②多動性** | 主体的な行動、多様なチャレンジ | 新しい挑戦、プロジェクト立ち上げ、実験 |
| **③感受性** | 内省、気づき、深い学び | 振り返り、フィードバック受容、自己理解 |

### 会社組織との補完関係

会社では以下が磨かれやすい：
- **組織性**：チームワーク、階層構造の中での動き方
- **一貫性**：計画通りの実行、ブレない姿勢
- **論理性**：合理的思考、データドリブンな判断

asotobaseはこれを**補完**し、バランスの取れた成長を支援します。

## 2. 解決したい課題

### 2.1 個人レベルの課題

#### 行動のハードル
- **課題**：あそとは「重要だが緊急でない」第2領域。多忙時は後回しになる
- **結果**：行動の一歩目が重い、継続できない
- **対策**：ゲーミフィケーション、小さなステップの可視化

#### 内省の不足
- **課題**：行動後に「やりっぱなし」になる
- **結果**：学びが定着しない、成長を実感できない
- **対策**：内省ログ、AIコーチングによる振り返り支援

### 2.2 コミュニティレベルの課題

#### 情報の分散・複雑さ
- **課題**：Discordでのやりとりが追いづらい
- **結果**：イベントやプロジェクトの全体像が見えない
- **対策**：統合ダッシュボード、情報の構造化

#### 関係性の不可視化
- **課題**：社外の仕事がご縁で回っているが体感しづらい
- **結果**：関係性の重要性を認識できない
- **対策**：関係性の可視化、つながりマップ

#### マッチングの属人化
- **課題**：「この人とこの人は合いそう」が特定の人の判断に依存
- **結果**：機会損失、スケールしない
- **対策**：データドリブンなマッチングアルゴリズム

#### 進捗の定性化
- **課題**：ビジョンや進捗が肌感・定性情報のみ
- **結果**：客観的な評価や改善が難しい
- **対策**：定量的な指標、ダッシュボード

## 3. asotobaseの目的

### 3.1 個人のあそと/自律を促進（②多動性 ③感受性）

#### あそとステップの可視化
- 行動を小さなステップに分解
- 進捗を視覚的に表示（プログレスバー、バッジなど）
- 達成感を積み重ねて継続を促す

#### ネクストアクションの整理
- システムが次の推奨アクションを提案
- 優先順位を明確化
- 「何をすればいいかわからない」を解消

#### 内省ログページ
- 個人専用の振り返りスペース
- 時系列での行動履歴
- タグ付け、カテゴリ分類

#### AIコーチング
- 行動に対する質問を投げかける
- 気づきを促すフィードバック
- 成長パターンの分析と提案

### 3.2 共創/共栄しやすくなる（①関係性）

#### スマートマッチング
- **人×人**：スキル、興味、価値観からマッチング
- **人×機会**：プロジェクト、イベント、仕事の推薦
- **人×企業**：副業、協業、転職機会の提示

#### 全体進捗の可視化
- コミュニティ全体のアクティビティ
- プロジェクトの進行状況
- メンバーの成長トレンド

#### 貢献度ポイント制
- イベント参加、プロジェクト貢献、他者支援などをポイント化
- ランキングではなく「共に成長している感」を演出
- ポイントに応じた特典やバッジ

#### コミュニティ活性化
- イベントカレンダー
- プロジェクトボード
- ディスカッションフォーラム

## 4. ユーザージャーニー

### 4.1 新規ユーザー（オンボーディング）

1. **登録**：プロフィール設定（スキル、興味、目標）
2. **診断**：あそと度チェック（現状の3要素スコア）
3. **目標設定**：短期・中期の「あそと目標」を設定
4. **推奨アクション**：最初の一歩を提案

### 4.2 アクティブユーザー（日常利用）

1. **ダッシュボード確認**：今日のおすすめアクション
2. **行動実行**：イベント参加、プロジェクト作業など
3. **ログ記録**：活動を記録（手動 or 自動）
4. **内省**：AIコーチングで振り返り
5. **つながり**：新しい人や機会と出会う

### 4.3 コミュニティリーダー

1. **全体把握**：ダッシュボードで全体状況を確認
2. **イベント企画**：新しいイベントを作成
3. **マッチング**：メンバー同士を紹介
4. **フィードバック**：メンバーの成長を支援

## 5. 成功指標（KPI案）

### 個人レベル
- **継続率**：30日後アクティブ率
- **行動頻度**：週あたりのアクション数
- **内省率**：行動に対する振り返り率
- **成長実感**：あそと度スコアの向上

### コミュニティレベル
- **アクティブメンバー数**
- **イベント開催数・参加率**
- **マッチング成立数**
- **コラボレーション創出数**
- **NPS（Net Promoter Score）**
